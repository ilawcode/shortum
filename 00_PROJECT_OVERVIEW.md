# ShortcutHub: AI-Powered iOS Shortcuts Ecosystem

## 1. Executive Summary
ShortcutHub is a web-based "No-Code" editor for Apple Shortcuts. Users can build, share, and fork native Apple actions through a drag-and-drop interface.

## 2. Core Principles
- **Style:** Glassmorphism & Minimalist UI.
- **Tech:** Next.js 15, Supabase, Tailwind CSS.
- **Safety:** OWASP Standards, Whitelist-only actions, UUID-based non-enumerable IDs.
- **Economy:** 10 Welcome Credits, -1 per project. Anti-cheat temp-mail blocking.

## 3. Communication
- Real-time in-app notifications (Supabase Realtime).
- Email notifications via Resend.
- Push-like Toast widgets for online users.
