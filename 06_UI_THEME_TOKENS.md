# Visual Design Tokens (Tailwind)

## 1. Colors
- **Primary:** Indigo-500 (Glass overlay).
- **Surface (Dark):** `rgba(15, 23, 42, 0.7)`.
- **Surface (Light):** `rgba(248, 250, 252, 0.6)`.
- **Border:** `rgba(255, 255, 255, 0.1)`.

## 2. Glassmorphism
- **Blur:** `backdrop-blur-xl`.
- **Saturation:** `backdrop-saturate-150`.
- **Shadow:** Subtle colored glow for active cards.
